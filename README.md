# CodeAlpha Email Extractor 📧

This project is a simple Python automation script created during the CodeAlpha Python Programming Internship.

The program reads a text file, extracts all email addresses using Regular Expressions (Regex), and saves them into another file automatically.

## 📌 Features

* Reads text from a file
* Extracts email addresses automatically
* Saves emails into a separate file
* Beginner-friendly Python project

## 🛠 Technologies Used

* Python 3
* Regular Expressions (re module)

## 📚 Concepts Used

* File Handling
* Regex Pattern Matching
* Loops
* Input/Output

## 📂 Files Included

* `email_extractor.py`
* `sample.txt`
* `emails.txt`

## ▶️ How to Run

1. Install Python on your system
2. Create a text file named `sample.txt`
3. Add sample email addresses inside the file
4. Run the Python program:

```bash id="f9y3de"
python email_extractor.py
```

## 📧 Example Input

```text id="4r7xmq"
hello@gmail.com
test@yahoo.com
company@outlook.com
```

## 📤 Output

The extracted email addresses will be saved in:

```text id="k1u6vn"
emails.txt
```
